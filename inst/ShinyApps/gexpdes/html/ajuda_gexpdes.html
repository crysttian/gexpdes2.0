Ajuda do GExpDes 2.0

 

1 - O que é o GExpDes?

Os dados gerados a partir de um experimento são analisados segundo metodologias de Estatística Experimental. Para realizar essas análises, destacam-se a utilização da linguagem R, particularmente com o pacote ExpDes. Para quem não quer se preocupar com a linguagem R, esse trabalho apresenta uma aplicação em Shiny que possibilita o uso dos pacotes ExpDes, labestData e outros pacotes voltados para Estatística Experimental. A interface reduz a complexidade da análise, sendo um auxiliar para a montagem dos comandos da linguagem R para realizá-la. Ao final, os comandos que foram necessários também são disponibilizados ao usuário.

5 - Como iniciar uma análise?

Para realizar uma análise, siga dos passos:

  1. Clique no botão "Iniciar a análise".;
  2. Selecione a origem dos dados e clique em Delineamento. Existem duas opções para a importação dos dados para análise: 
    1. Importar os dados de um arquivo;
    2. Importar os dados da biblioteca LabestData.
  3. Selecione o delineamento desejado e complete as informações relativas ao experimento. Após configurar o delineamento, clique em Análise;
  4. Selecione os demais testes e opções do delineamento e visualize as análises apresentadas nas guias :
    1. Análise Preliminar
    2. Análise de Resíduo
    3. Análise de Variância
  5. Caso deseje realizar uma nova análise, clique em "Nova Análise" ou caso deseje fechar a aplicação, clique em "Fechar".

6 - Quais são os formatos de arquivo que o GExpDes importa?

Atualmente, o GExpDes permite a importação de arquivos nos formatos CSV (recomendado), planilha eletrênica (Excel ou LivreOffice ou OpenOffice). 

7 - Como posso salvar as minhas análises?

Na guia de Análise de Variância é possível exportar a análise gerada. As demais figuras, caso seja necessário, existem duas opções, clicando no ícone de uma câmera ou clicando sobre a imagem e escolhendo a opção "Salvar imagem como".

8 - O que devo fazer quando identificar um erro no GExpDes?

Em nossos testes, a maioria dos erros detectados ou nos relatados, estão relacionados na estrutura dos arquivos de dados. Solicitamos que faça uma verificação da estrutura do seu arquivo.

Caso identifique um erro, pedimos para nos enviar um e-mail gexpdes@gmail.com ou nos envie uma mensagem pelo grupo do Telegran direto para os desenvolvedores. Para entar no grupo do Telegran acesse o qrcode na interface ou no site do projeto.
