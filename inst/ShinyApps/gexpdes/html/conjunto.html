Origem do conjunto de Dados

Existem três possibilidades para carregar os seus dados para o GExpDes, sendo:

  • Importação de arquivo de dados: permite realizar a importaçaõ de um conjunto de dados diretamente de um arquivo nos formatos Comma/Tab-separated data (CSV, TSV), Excel (XLS/XLSX) e Open/Libre Office (ODS).
  • Dados da Biblioteca LabestData: permite importar um conjunto de dados diretamente da biblioteca LabestData
  • Gerar os arquivos de dados: permite gerar um conjunto de dados a partir da configuração de um delineamento
